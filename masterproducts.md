## Database Maintenance Notes

### When adding products:
- ‚úÖ Check for duplicates first
- ‚úÖ Use consistent naming (e.g., "Bombay Sapphire Gin 70cl" not "Bombay Gin 700ml")
- ‚úÖ Include multiple sizes where appropriate (25ml, 50ml, 70cl, 1L for spirits)
- ‚úÖ Use correct unit_size in ML (700 not "70cl", 750 not "75cl")
- ‚úÖ Set case_size to realistic distributor packs (6, 12, 24, etc.)
- ‚úÖ Use unit_type: 'bottle', 'can', 'keg', etc.

### Removal strategy:
- ‚ùå Don't delete products (keeps historical data)
- ‚úÖ Use soft delete: set `active = false` if needed

### Query to find what's been added:
```sql
SELECT category, subcategory, COUNT(*)
FROM master_products
WHERE active = true
GROUP BY category, subcategory
ORDER BY category, COUNT(*) DESC;
```
