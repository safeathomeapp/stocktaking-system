## Database Maintenance Notes

### When adding products:
- ✅ Check for duplicates first
- ✅ Use consistent naming (e.g., "Bombay Sapphire Gin 70cl" not "Bombay Gin 700ml")
- ✅ Include multiple sizes where appropriate (25ml, 50ml, 70cl, 1L for spirits)
- ✅ Use correct unit_size in ML (700 not "70cl", 750 not "75cl")
- ✅ Set case_size to realistic distributor packs (6, 12, 24, etc.)
- ✅ Use unit_type: 'bottle', 'can', 'keg', etc.

### Removal strategy:
- ❌ Don't delete products (keeps historical data)
- ✅ Use soft delete: set `active = false` if needed

### Query to find what's been added:
```sql
SELECT category, subcategory, COUNT(*)
FROM master_products
WHERE active = true
GROUP BY category, subcategory
ORDER BY category, COUNT(*) DESC;
```
